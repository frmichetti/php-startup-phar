# php-startup-phar
Getting Started with PHP phar Files

### How To Use

1. Clone this Project

```sh
$ git clone [git-repo-url]
$ cd php-startup-phar
```
2. Put Your Files in src/

3. Third Make the phar File

```sh
$ php create-phar.php
```

4. Run The App

```sh
$ cd run
$ php -S localhost:8000
```
